# Cotesa Tours Kenya

A modern, responsive website for Cotesa Tours, showcasing authentic Kenya safari experiences.

## Features

- Fully responsive design
- Smooth animations and interactions
- Custom cursor
- Tour filtering
- Functional contact form with backend
- Gallery with hover effects
- Testimonials carousel
- SEO optimized

## Structure

- `index.html` - Main HTML file
- `style.css` - All styles
- `script.js` - JavaScript functionality
- `server.js` - Express server
- `package.json` - Dependencies

## How to Run

1. Install dependencies: `npm install`
2. Start the server: `npm start`
3. Open `http://localhost:3000` in your browser

## Technologies Used

- HTML5
- CSS3 (Custom properties, Grid, Flexbox)
- Vanilla JavaScript
- Express.js
- Google Fonts (Cormorant Garamond, DM Sans)

## Images

Images are sourced from Unsplash. For production, consider optimizing and hosting locally.

## Contact Form

The contact form now submits to the backend server. Form data is logged to console (in production, integrate with email service).
